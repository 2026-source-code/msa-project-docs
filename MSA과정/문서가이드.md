# 문서 가이드

## 단일 진입점: 홈페이지 대시보드

> **작성은 각자 편한 곳에서, 확인은 홈페이지 한 곳에서!**

| 도구 | 용도 | 누가 | 대시보드 연결 |
|------|------|------|-------------|
| **홈페이지** | 개인평가 + 프로젝트평가 → 서류 자동 생성 + **대시보드** | 강사 | PDF 직접 보기/다운로드 |
| **노션** | 프로젝트 문서 10종 작성 | 학생 | 노션 링크로 연결 |
| **GitHub** | 코드 + 스텝 완료시 master 커밋 → 강사 코드리뷰 | 학생 | GitHub 링크로 연결 |

팀 등록 시 **노션 URL**과 **GitHub URL**을 홈페이지에 입력하면,
대시보드에서 15종 서류를 한 화면으로 전부 확인할 수 있습니다.

---

## 1. 평가/채점/평가문서 관리

### 평가 종류

| 구분 | 평가 내용 | 도구 | 시기 |
|------|----------|------|------|
| 개인 포트폴리오 평가 | GitHub + 노션 근거자료 기반 | 평가사이트 | 중간/최종 |
| 개인 객관식 평가 | 기술 지식 평가 | 평가사이트 | 중간/최종 |
| 프로젝트 평가표 | 6개 영역 100점 만점 채점 | 평가사이트 | 중간/최종 |
| 관찰일지 | 대인관계/문제해결 능력 (5점) | 평가사이트 | 주 1회 |
| 멘토링일지 | 멘토링 내용/피드백 기록 | 평가사이트 | 멘토링 후 (4회) |
| 동료평가 | 팀원 간 상호 평가 | 노션 | 중간/최종 |

### 강사 프로세스 (평가)

```
주간 루틴
──────────────────────────────────────
매주 1회 → 평가사이트에서 관찰일지 입력
            (학생별 대인관계/문제해결 점수 + 한줄 코멘트)
            근거: 노션 회의록/하브루타 + GitHub 커밋/PR 확인

멘토링 후 → 평가사이트에서 멘토링일지 입력
            (질문, 답변, 피드백, 액션아이템)

중간/최종  → 평가사이트에서 프로젝트 평가표 입력
            (6개 영역 점수 + 총평)
          → 포트폴리오 평가, 객관식 평가 진행

           ↓ 자동 생성

         채점지, 총평, 체크리스트,
         관찰일지, 평가표, 멘토링일지 PDF
```

### 평가사이트에 추가할 입력 폼

**관찰일지 폼** (주 1회)

| 입력 항목 | 타입 |
|----------|------|
| 관찰 차수 | 선택 (1차~4차) |
| 팀 선택 | 드롭다운 |
| 학생별 대인관계 능력 | 1~5점 |
| 학생별 문제해결 능력 | 1~5점 |
| 학생별 관찰 내용 | 텍스트 |
| 근거자료 체크 | 체크박스 (GitHub, 회의록, 하브루타, 블로그, 스탠드업) |
| 종합 소견 | 텍스트 |

**평가표 폼** (중간/최종)

| 입력 항목 | 타입 |
|----------|------|
| 평가 유형 | 중간/최종 선택 |
| 팀 선택 | 드롭다운 |
| 계획 (20점) | 3개 항목 점수 + 근거 |
| 내용 (20점) | 3개 항목 점수 + 근거 |
| 수준 (15점) | 2개 항목 점수 + 근거 |
| 기능수준 (15점) | 3개 항목 점수 + 근거 |
| 태도 (10점) | 3개 항목 점수 + 근거 |
| 구두발표 (20점) | 3개 항목 점수 + 근거 |
| 총평 6개 항목 | 텍스트 |

**멘토링일지 폼** (4회)

| 입력 항목 | 타입 |
|----------|------|
| 멘토링 차수 | 선택 (1차~4차) |
| 멘토 정보 | 이름, 소속, 유형(대면/비대면) |
| 대상 팀 | 드롭다운 |
| 질문/답변 | 반복 입력 (번호, 질문, 답변) |
| 멘토 피드백 | 잘한점, 개선점, 추천자료 |
| 액션 아이템 | 반복 입력 (내용, 담당자, 기한) |

## 2. 프로젝트 PBL 과제 관리

### 학생 작성 문서 9종 (노션)

| 문서 | 작성 단위 | 작성 시기 | 제출 근거 |
|------|----------|----------|----------|
| 요구사항정의서 | 팀 | 1주차 | MVP, DoD, 기능/비기능, ERD, API |
| 스프린트계획서 | 팀 | 스프린트마다 | 백로그, 진행률, KPT |
| 회의록 | 팀 | 매일 | 안건, 사진, 스탠드업 |
| 팀블로그관리 | 팀 | 수시 | 블로그 현황, 발표기록, 노션 복제 |
| 코드리뷰 | 팀 | 스텝 완료시 | master 커밋 강사 리뷰 결과 기록 |
| 트러블슈팅 | 팀 | 에러 발생시 | 증상→원인→해결→배운점 |
| 프로젝트회고 | 팀 | 종료 시 | KPT, 팀원별 회고 |
| 하브루타노트 | 개인 | 주 2~3회 | What/Why/Problem/Solution/회고 |
| 동료평가 | 개인 | 중간/최종 | 팀원별 4개 항목 점수 |

### 학생 프로세스

```
프로젝트 시작
│
├─ 1주차: 요구사항정의서 작성 (노션)
│         팀 구성, MVP/DoD 정의, ERD, API 설계
│
├─ 매일:  회의록 작성 + 회의 사진 (노션)
│         하브루타노트 작성 (노션, 개인)
│         코드 작업 (GitHub, feature 브랜치)
│
├─ 스텝 완료시: master 커밋 → 강사 리뷰 → 리뷰 결과 기록 (노션)
│
├─ 에러시: 트러블슈팅 기록 (노션)
│
├─ 스프린트마다: 스프린트계획서 + KPT 회고 (노션)
│
├─ 수시:  블로그 작성 → 노션 복제
│         주 1회 블로그 발표
│
├─ 중간:  동료평가 (노션, 개인)
│
└─ 종료:  프로젝트회고 (노션)
          동료평가 최종 (노션, 개인)
          모든 문서 정리
```

### 강사 프로세스 (PBL 운영)

```
프로젝트 시작 전
│
├─ PBL 계획서 작성 (이미 완료)
├─ 요구사항 가이드 제시 (이미 완료)
├─ [홈페이지] 팀 등록 (노션 URL + GitHub URL 입력)
│
프로젝트 진행 중
│
├─ 매일:  스탠드업 참관, 질의응답 대응
│         [대시보드 → 노션/GitHub 링크] 근거자료 확인
├─ 스텝 완료시: GitHub master 커밋 코드리뷰
├─ 주 1회: 관찰일지 입력 (홈페이지)
├─ 멘토링 후: 멘토링일지 입력 (홈페이지)
│
├─ 중간 발표: 평가표 입력 (홈페이지)
│
└─ 최종 발표 / 국가 제출
    ├─ 평가표 입력 (홈페이지)
    └─ [대시보드] 15종 서류 한눈에 확인 → PDF 일괄 다운로드
```

## 3. 제출 체크리스트 (대시보드에서 한눈에)

> 홈페이지 대시보드에서 아래 15종 서류를 **링크 + PDF**로 전부 확인 가능

| 번호 | 서류 | 작성 | 작성 도구 | 대시보드에서 보기 |
|-----|------|------|----------|-----------------|
| 1 | PBL 프로젝트 계획서 | 강사 | 홈페이지 | [PDF 보기] |
| 2 | 요구사항 정의서 | 학생 | 노션 | [노션 열기] |
| 3 | 관찰일지 (주차별) | 강사 | 홈페이지 | [PDF 보기] |
| 4 | 평가표 (중간/최종) | 강사 | 홈페이지 | [PDF 보기] |
| 5 | 동료평가표 | 학생 | 노션 | [노션 열기] |
| 6 | 하브루타노트 | 학생 | 노션 | [노션 열기] |
| 7 | 회의록 + 회의사진 | 학생 | 노션 | [노션 열기] |
| 8 | 스프린트계획서 | 학생 | 노션 | [노션 열기] |
| 9 | 멘토링일지 | 강사 | 홈페이지 | [PDF 보기] |
| 10 | GitHub 주소 | 학생 | GitHub | [GitHub 열기] |
| 11 | 팀블로그 + 노션 복제 | 학생 | 노션 | [노션 열기] |
| 12 | 프로젝트 회고록 | 학생 | 노션 | [노션 열기] |
| 13 | 트러블슈팅 문서 | 학생 | 노션 | [노션 열기] |
| 14 | 코드리뷰 기록 | 학생 | 노션 | [노션 열기] |
| 15 | 최종 발표 자료 | 학생 | 노션 | [노션 열기] |

**강사 직접 작성**: 1번 (이미 완료)
**홈페이지 자동생성**: 3, 4, 9번 (점수만 입력 → PDF)
**학생 노션 작성**: 나머지 전부 (대시보드에서 링크로 연결)
